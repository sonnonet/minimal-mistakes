# RaspberrySensorKit 설치

## 이미지 다운로드
```
- http://cogcom.asuscomm.com:6080/open/2017-06-sonno-raspi.zip
- 압축해제후 SD카드에 이미지 Write
```
## 기본 설치 프로그램

### 데이터 베이스
- Hbase-1.0.1.1
- influxdb-1.2.0

### 웹 GUI SW
- OpenTSDB-1.2.0
- grafana-4.1.2

### 임베디드 SW
- tinyos.2.1.2

### Middleware 
- serial_win10_tpp.py
- WSN Data Collection Python 
- OpenTSDB Python Driver
- influxdb Python Driver
